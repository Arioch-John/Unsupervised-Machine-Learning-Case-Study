**MENTAL HEALTH IN TECH RELATED JOBS**

---

- This Case Study is based on the OSMI Mental Health in tech 2016 survey.
- It focus on using **Unsupervised Machine Learning** Techniques and analysing the mental health of employees.

- The objective of this project is to identify groups of employees with similar characteristics and experiences regarding mental health and workplace conditions.
- Along with this it helps to show the skills I developed for methods such as Data Cleaning, Feature Engineering and Clustering.
- There are also visualisations present in the notebook to help with understanding the data and clusters.
---
- Several Libraries such as NumPY, Pandas, Seaborn, Matplotlib, and Scikit-Learn were used.
- Python version used for this project is 3.13.9
---
- The final file that was made is the USML_Case_Study.ipynb; however, several alternate approaches were also considered which can be found in the Try out files directory.
---
